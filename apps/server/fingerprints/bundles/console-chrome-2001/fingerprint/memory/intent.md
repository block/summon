# Console Chrome 2001 Intent

This fingerprint captures a source-agnostic Y2K console-web composition language inspired by public historical study of Nintendo.com circa 2001 and adjacent retro game-portal aesthetics. It is not a Nintendo brand kit.

Generated surfaces should feel like compact game-console hardware rendered as UI: periwinkle molded chrome, carbon command slabs, amber/orange wayfinding, dense modules, small uppercase labels, outlined box-art hero type, halftone texture, and hard bevel seams.

The canonical intent is to preserve the reusable interface grammar without reusing protected source material. Do not include Nintendo-owned marks, characters, screenshots, console product names, exact navigation labels, exact copy, or regulated rating marks unless a user explicitly supplies them as content for a specific transformation. Prefer original fictional products, generic mascots, abstract cartridge/controller motifs, and prompt-derived copy.

Modern adaptations may improve accessibility and narrow-screen usability while preserving the faceplate metaphor: controls can have larger invisible hit areas, rails can stack below content, and command bars can collapse, but the surface should still read as one assembled machine rather than a set of generic cards.
