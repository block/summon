## Composition

A comparison helps choose between low-stakes personal options. Reach for this
surface when the user's first question is "which of these should I go with — and
it's not a big deal either way?" — comparing errands, purchases, appointments, or
household choices on human, practical grounds — not when one next move is already
clear (that is the [planner](../planner)), when the task is a sequence to move
through (that is the [staged-plan](../staged-plan)), or when it is a repeatable
routine (that is the [routine](../routine)).

**Fit/friction/timing comparison.** Life-admin comparisons use fit, friction,
timing, materials, confidence, and emotional load — human, practical criteria
instead of false precision. Use soft columns or stacked [note panels](../note-panels)
on `--garden-note-paper` rounded to `--garden-panel-radius` rather than a dense
ruled matrix, and keep the criteria human and parallel across
each option so the reader weighs like against like. This is a gentle variant of the
note-panel system: where a plain plan holds one path, a comparison holds a few
options weighed on shared soft criteria — never a numeric scorecard or a
monochrome critique grid.

**A kind recommendation.** Give the recommended option a warmer note marked with a
single `--garden-chosen-dot` (sized at `--garden-chosen-marker`, haloed by
`--garden-chosen-ring`) or a friendly `--garden-chip-radius` [badge](../badges) —
never a severe winner label, a hard red/green winner-and-loser, or a colorful score
system — and name the accepted tradeoff in one sentence through the
[good-enough choice note](../good-enough). The green dot lands on exactly one option;
any "check first" caution on another rides `--garden-clay-accent`, not alert red. The recommendation is a kind nudge
toward a good-enough move for this moment, not a verdict; avoid hard winner/loser
language when stakes are personal and low.

End by making the recommended option easy to act on with a concrete verb the user
can do today, and name what can wait if some options need not be decided now.
