## Composition

A planner makes the next small step feel doable now. Reach for this surface when
the user's first question is "what is the easiest next thing I can do right now?" —
a messy personal request that needs a single clear first move with its context held
nearby — not when the task has real sequence that needs staging (that is the
[staged-plan](../staged-plan)), when the job is to weigh options (that is the
[comparison](../comparison)), or when it is a repeatable routine (that is the
[routine](../routine)).

**Planner note stack.** Build a bounded note stack inside the soft
[note panels](../note-panels): a context header that reflects the real task, one
visibly primary next small step, and supporting notes for prep, constraints, and
reminders arranged around it. Open with a short context line that reflects the real
task; never lead with introductory advice before the first action. The stack sits on
`--garden-note-paper` rounded to `--garden-panel-radius`; keep the surface airy but
bounded so it never feels empty, and let exactly one primary step carry the
`--garden-chosen-dot` (sized at `--garden-chosen-marker`, haloed by
`--garden-chosen-ring`) in a soft rounded callout so the eye lands on what to do
now. The green marker appears once — if it shows up twice it has stopped meaning
"start here."

**Now/next/later only when needed.** Now/next/later is a pressure-release device,
not a required template. Use it when the task has sequence or overload, skip it for
tiny one-step tasks, and make Later explicitly reassuring when it appears — never
hide the actual first action under process labels. When the task is a single move,
one note is enough.

The primary action and any can-wait framing follow the
[good-enough choice and can-wait notes](../good-enough); readiness, effort, and
energy ride on `--garden-chip-radius` [badges](../badges) faced with
`--garden-chip-face` and inked in `--garden-chip-ink`. The whole stack must end in a concrete verb
the user can do today — gather, pick, check, text, set aside, start, wait — and
never a productivity verb.
