# Technical Contrast Core Examples

These are agnostic, derived visual exemplars for the Technical Contrast
fingerprint. They are intentionally not screenshots of the researched source
website and do not include source logos, names, domains, slogans, customer marks,
exact model/pricing values, proprietary screenshots, or proprietary artwork.

- `technical-plane-landing.png` — pale technical landing plane with claim, mono
  actions, proof rail, and optional signal artwork.
- `white-technical-pricing-sheet.png` — white comparison sheet with left category
  rail, segmented tabs, pale table header, and dense aligned rows.
- `midnight-research-proof-plane.png` — midnight proof plane with display claim,
  recognition rail, and dark-on-dark proof cards.
- `technical-config-workflow.png` — white form/configuration sheet with mono
  labels, hairline grouping, and one black primary action.
- `code-breakout-capability-panel.png` — white technical plane with a minimal
  dark code/configuration breakout and proof facts.
- `mobile-technical-landing-stack.png` — narrow-screen landing stack with a
  claim-first hero, compact rectangular CTAs, large cropped signal artwork, and
  muted proof rail.
- `technical-benchmark-update-card.png` — source-neutral technical update card
  row with mono category badges, bold headlines, neutral graphic panels, and
  implication copy.

Treat these images as weighted or supporting source-informed evidence pending
scored dogfood review, not as canonical brand assets.
