# Technical Contrast Intent

Technical Contrast is an agnostic technical-platform surface language distilled
from public AI-infrastructure web research and the attached design analysis. It
is not a clone of the source website and must not reuse source names, domains,
logos, slogans, literal wordmarks, proprietary copy, screenshots, or hero art.

The durable language is not a decorative effect. It is the combination of
contrast planes, technical type contrast, white data sheets, midnight proof
planes, hairline structure, compact comparison tables, and small-radius
rectangular controls. Optional signal artwork may appear at large scale on a
landing or proof plane, but it is subordinate to the claim, data, and action.
Data-heavy and workflow-heavy surfaces should omit artwork when it would reduce
clarity.

Use this fingerprint to generate surfaces that feel precise, production-ready,
and infrastructure-aware without becoming generic dark SaaS, neon AI branding,
or friendly rounded-card software. The best output should make technical choices
and proof easier to evaluate: the claim is visible first, shared criteria stay
aligned, proof explains why the claim is credible, and the next action is clear.

The included examples are derived neutral exemplars created for Ghost curation.
They are not source screenshots and should be treated as composition evidence,
not brand assets.
