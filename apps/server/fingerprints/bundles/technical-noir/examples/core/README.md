# Technical Noir Core Exemplars

These portable exemplars are agnostic Technical Noir references created from source research, not copied source-site screenshots.

## Exemplars

- `technical-landing-proof.png` — quiet technical landing composition with command action and proof panel.
- `agent-workspace-overview.png` — product/workspace shell with task rail, evidence, execution plan, and next action.
- `operational-comparison.png` — shared-criteria comparison with off-white recommendation and terminal estimate evidence.

## Curation stance

The public source research informed composition moves: claim followed by proof, product UI as evidence, command snippets near actions, tab/selector plus evidence frame, and compact setup tiles. These exemplars deliberately avoid source product names, logos, partner/customer names, exact copy, screenshots, and proprietary claims.

Treat these as draft portable exemplars until human review accepts or replaces them. The editable generation source is retained in `source.html` for future curation and regeneration.
